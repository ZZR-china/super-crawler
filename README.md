## Node Express Mongoose

A boilerplate application for building web apps using express, mongoose and passport.


## Usage

    $ git clone https://github.com/ZZR-china/template-express-mongoose.git
    $ cd template-express-mongoose
    $ yarn install
    $ yarn start
